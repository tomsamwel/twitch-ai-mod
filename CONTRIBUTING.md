# Contributing

This is a personal project, but contributions are welcome via issues and pull requests.

## Setup

See the [README](README.md) for local setup instructions.

## Before Submitting

1. Run `npm run check` (TypeScript type check)
2. Run `npm test` (unit tests)
3. Follow existing code style: TypeScript strict mode, no `any`, ESM imports

## Reporting Issues

Open a GitHub issue with steps to reproduce. If it involves AI behavior, include the relevant scenario YAML or chat context.
